# Полноценная адаптив верстка сайта "Сложно сосредоточиться" с опцией переключения тем.
## Были использованный медиазапросы для 3 разрешений:  
* 375x668 (стартовое разрешение);
* 768x1024;
* 1024x768;
## Две разные цветовые темы сайта, а также тема auto:
На сайте присутсвуют выбор цветовых тем. По умолчанию тема стоит *auto* и это означает, что цветовая тема у сайта будет адаптироватся под настройки цветовой темы пользователя.
